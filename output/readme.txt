These data files contain the "output" of the instances considered in
S. Martello, M. Monaci, D. Vigo
"An Exact Approach to the Strip-Packing Problem"

The format of each of these data files is:
height of the strip
unused space of the strip
for each piece i (i=1,...,n): 
  index i, cartesian x position x_i , cartesian y position y_i and rotation {bool}  of item i.
